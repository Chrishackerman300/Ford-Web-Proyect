Proyecto Pagina web de Ford

En este proyecto haremos desde cero una pagina web sobre ford.

Donde utilizaremos muchas propiedades tanto de HTML y CSS, en html usaremos metodologías muy utilizadas en el desarrollo web como lo es (BEM)

En CSS utilizaremos flexbox, media queries, varaibles, pesudoelementos, pseudoclases y más!

La pagina contara con un darkmode para que el usuario lo active cuando quiera y tenga mas interactividad al igual crearemos un sidebar menu. 

Que trandrá acciones u eventos con JavaScript donde le daremos una logica de programción para que mande a llamar a los eventos de la web, una vez
que el usuario interactue con ellos.
